O que é o HUB SIGeo?
O HUB SIGeo foi desenvolvido para apresentar informações gerais sobre o nosso trabalho, compilar as aplicações que são elaboradas pela nossa equipe em uma única plataforma, além de disponibilizar os dados do munícipio de Niterói de forma transparente ao usuário. Toda a base geográfica municipal disponível pode ser acessada na seção Dados Abertos.
Qual a fonte dos dados disponibilizados no site?
Os dados dispostos na seção Dados Abertos são de responsabilidade de diferentes secretarias, sendo estas encarregadas pela elaboração e atualização dos mesmos. O objetivo do SIGeo consiste, portanto, em fazer o gerenciamento e a manutenção dessas informações no banco de dados para disponibilizá-las ao público.
Como os dados são coletados?
A maior parte dos dados disponíveis são adquiridos via fonte administrativa, ou seja, por meio das secretarias, departamentos e demais estruturas que compõem a Prefeitura de Niterói. Os dados enviados por essas fontes passam por um processo de geocodificação dos endereços, de modo a atribuir coordenada geográfica para cada ponto. Além disso, também coletamos dados relacionados ao município de Niterói de fontes externas. Caso queira saber sobre a coleta de algum dado específico, entre em contato com a secretaria responsável. Você consegue essa informação através da planilha de metadados das camadas clicando aqui.
Com que frequência os dados e informações dispostas aqui são atualizadas?
Os dados disponibilizados aqui são atualizados à medida que novos estiverem disponíveis, portanto, não há uma periodicidade exata. Para garantir que essa verificação seja feita sempre que possível, procuramos manter constante contato com as secretarias responsáveis para que essas mantenham essas informações atualizadas no nosso banco de dados.
Preciso de um login para utilizar o Geoportal?
Você pode acessar as camadas e informações públicas clicando no botão de “Acesso Público”. A criação de login de acesso é exclusivo para servidores da Prefeitura de Niterói mediante solicitação através do nosso e-mail de atendimento.
Como posso usar o conteúdo aqui disposto?
O HUB SIGeo fornece dados e análises importantes que agências governamentais, organizações com e sem fins lucrativos, pesquisadores, estudantes, grupos comunitários e o público podem usar em seu trabalho, planejamento e processo de tomada de decisão. Os dados e a análise podem ajudar na produção de estudos ambientais, viabilidade econômica, crescimento demográfico, entre outros que podem ser usados para determinar alocações de recursos com base na necessidade.
Mais dados podem ser adicionados?
Novos dados podem ser disponibilizados à medida que são produzidos ou demandados pelas secretarias do município e que tenham interesse público. Também podemos disponibilizar novas informações baseadas no retorno do usuário, incluindo solicitações de dados ou funcionalidades adicionais. Você pode nos enviar esse feedback através da nossa pesquisa de satisfação clicando aqui.
Como posso solicitar informações que não encontrei no site?
Entre em contato conosco através do nosso e-mail atendimento@sigeo.niteroi.rj.gov.br para que possamos atendê-lo da melhor forma.
